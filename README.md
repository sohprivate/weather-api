# Weather API

複数の天気予報APIを統合して、より信頼性の高い天気予報を提供するシステムです。

## 機能

- 複数の天気予報APIからのデータ取得
- データの正規化と比較
- 最適な予報の選択

## セットアップ

1. 依存パッケージのインストール:
```bash
pip install -r requirements.txt
```

2. 環境変数の設定:
`.env`ファイルを作成し、必要なAPIキーを設定してください。

## 使用方法

```bash
python run_weather_graph.py
```
